FindAndReplaceThenCopy 
==========================================DEBUT UTILISATION NORMAL IFIX=================================================================
Ce programme est fait pour être utiliser avec le logiciel SCADDA IFIX 

Il permet de cree un grande quantite de tag selon une structure de base que nous vonllons repeter a plusieur reprise pour different PLC.

1. Cree une premiere bach de tag que nous voulons pour 1 seul PLC dans SCADDA IFIX
2. Exporter c'est tag de SCADA IFIX (fichier.csv)
3. Convertir se (fichier.csv)==>(fichier_entree.tx) le nom dois être "fichier_entree.txt"
4. Ouvrir votre fichier, il de vrai ressembler a cel : ouvrir (fichier_entree_AVANT_modification.txt)
5. Modifier le fichier pour qu'il ressemble a cela : ouvrir (fichier_entree_APRES_modification.txt)
6. Pour comprendre la structure ouvrir se fichier txt : ouvrir (fichier_entree_APRES_modification_EXPLICATION.txt)
7.Mettre le fichier text dans le repertoire de debug avec le executable
8.clicker sur l'executable et vous aurez un fichier text de sortie : exemple ouvrir (fichier_sortie_example.txt)

Voila pour l'utilisation normal
